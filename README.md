## Compile it on your system

1. Clone the repo: `git clone https://github.com/nullspace1/2024-1C-3051-fossers.git`.
2. cd into the parent folder: `cd 2024-1C-3051-fossers`.
3. Build: `dotnet build`.
4. Run: `dotnet run`.
